# Projet : Emploi des Jeunes Diplômés en Europe

## Contexte
Analyse de l'emploi des jeunes diplômés en Europe à partir des données Eurostat.
Objectif principal : comprendre l'impact de l'IA sur l'emploi des jeunes.

## Utilisateur
Mohamed — professionnel data, expertise Power BI.

## Environnement
- OS : Windows 11
- Python installé (PowerShell comme shell)
- Données : Eurostat

## Stack technique
- Python + Pandas pour le traitement des données
- Plotly pour les visualisations interactives
- Rapports HTML style Power BI (interactif, export autonome)

## Conventions
- Rapports en HTML autonome (tout embarqué, pas de dépendances externes)
- Visualisations Plotly : couleurs sobres, style dashboard BI
- Code Python concis, sans commentaires superflus
- Réponses directes et concises
